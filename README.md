# recordtec-fe

Vue.js frontend for cloud networking test.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Configuration

The backend API URL is hardcoded in `src/App.vue`. Change the `API_URL` constant to point to your REST backend when ready. By default it uses a placeholder API for testing.
